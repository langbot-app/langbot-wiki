---
title: "スキルツール API"
---

Skills システムは Agent に 2 つのツールを提供します。どちらもサンドボックスバックエンドが利用可能である必要があります。Box が利用できない場合、Agent には公開されません。

| ツール | 用途 |
|------|------|
| `activate` | 既存の Skill を起動し、SKILL.md をコンテキストに取り込む |
| `register_skill` | サンドボックスに用意したディレクトリを新しい Skill として登録 |

---

## `activate`

### パラメータ

| パラメータ | 型 | 必須 | 説明 |
|------|------|------|------|
| `skill_name` | string | はい | 起動する Skill の名前 |

### 動作

1. skill store から該当名を検索
2. SKILL.md の内容を Tool Result として返し、現在の会話コンテキストに注入
3. その Skill を起動済みとして記録し、`/workspace/.skills/{skill_name}/` 経由でアクセス可能にする

起動後:
- `read` / `glob` / `grep` はマウントされたすべての Skill を読める
- Skill パッケージに対する `exec` / `write` / `edit` は **起動済み** の Skill に限られる

### 呼び出し例

```json
{ "skill_name": "pdf" }
```

### 戻り値

```json
{
  "activated": true,
  "skill_name": "pdf",
  "mount_path": "/workspace/.skills/pdf",
  "content": "<command-message>The \"pdf\" skill is activated</command-message>\n<skill-activation>\n  ...SKILL.md の内容...\n</skill-activation>"
}
```

### エラー

- `skill_name is required` — 名前が提供されていない
- `Skill "xxx" not found` — Skill が存在しない

## `register_skill`

`/workspace` 配下に用意したディレクトリを skill store に登録します。

### パラメータ

| パラメータ | 型 | 必須 | 説明 |
|------|------|------|------|
| `path` | string | はい | サンドボックス内のディレクトリパス。`/workspace` 配下である必要がある |
| `name` | string | いいえ | Skill 名を上書き（デフォルトは SKILL.md の `name` またはディレクトリ名） |
| `display_name` | string | いいえ | 表示名を上書き |
| `description` | string | いいえ | 説明を上書き |
| `instructions` | string | いいえ | 指示本文を上書き |

### 前提条件

- サンドボックスが有効かつ利用可能
- `path` が `/workspace` 配下
- そのディレクトリ自身、または 2 階層以内のサブディレクトリに `SKILL.md` が存在

### 典型的なフロー

```
1. Agent がサンドボックス内にディレクトリを準備:
   /workspace/my-skill/
   ├── SKILL.md
   └── scripts/main.py

2. Agent が register_skill(path="/workspace/my-skill") を呼び出す

3. Skill が Box Runtime の skill store に書き込まれ、
   以降どのセッションからでも activate で利用可能になる
```

### 呼び出し例

```json
{
  "path": "/workspace/my-skill",
  "name": "data-processor",
  "description": "CSV/JSON データを処理・変換"
}
```

### 戻り値

```json
{
  "registered": true,
  "skill_name": "data-processor",
  "source_path": "/workspace/my-skill",
  "skill": {
    "name": "data-processor",
    "display_name": "データ処理ツール",
    "description": "CSV/JSON データを処理・変換",
    "package_root": "/.../skills/data-processor"
  }
}
```

### エラー

- `path is required` — パスが提供されていない
- `path must be under /workspace` — パスがワークスペース内にない
- `Directory does not exist` — ディレクトリが存在しない
- `No SKILL.md found in ...` — ディレクトリおよび 2 階層以内のサブディレクトリに `SKILL.md` がない
- `Multiple skill directories found...` — パス配下に複数の Skill ディレクトリが存在する。`path` を具体的なサブディレクトリに指定すること
